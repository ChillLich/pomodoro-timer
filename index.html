🍅 Pomodoro Timer

Фокус. Работа. Отдых.

Простой и гибкий таймер Pomodoro на Python с Tkinter. Настраиваемые темы, звуки и управление медиа для максимальной продуктивности.

⬇ Скачать для Windows (.exe) 🐙 Исходный код (GitHub)

Возможности

⏱

Умные циклы

Автоматический подсчет сессий до длинного перерыва. Фазы: Focus, Short Rest, Long Rest, Pause.

🎨

Темы оформления

Поддержка светлой, темной и пользовательских тем. Настройка цветов фона, кнопок и текста.

🎵

Звуки и Медиа

Выбор своих MP3 треков для начала фазы. Управление системным медиаплеером (Play/Pause), он может управлять воспроизведением во многих браузерах и плеерах.

⚙️

Гибкие настройки

Пресеты времени (Small, Medium, Big или пользовательский), режим "Поверх всех окон", автосохранение конфигурации.

Установка

1. Скачайте последний .exe файл со страницы Releases.
2. Запустите файл.
3. Готово!

Ссылка: github.com/ChillLich/pomodoro-timer/releases 

# 1. Клонируйте репозиторий
git clone https://github.com/ChillLich/pomodoro-timer.git
cd pomodoro-timer

# 2. (Рекомендуется) Создайте виртуальное окружение
python -m venv venv
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# 3. Установите зависимости
pip install -r requirements.txt

# 4. Запустите
python gui.py
